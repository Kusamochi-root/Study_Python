## Python Basic Commands

| Command | Example | Description |
|---------|---------|-------------|
| print() | print("Hello") | Output text |
| input() | input("Enter: ") | Get user input |
| type() | type(10) | Check data type |
| len() | len("abc") | Get length |
| range() | range(5) | Create number sequence |
| int() | int("10") | Convert to integer |
| float() | float("3.14") | Convert to float |
| str() | str(123) | Convert to string |

---

## Control Statements (if / for / while)

| Statement | Example | Description |
|-----------|---------|-------------|
| if | if x > 0: | Conditional branch |
| for | for i in range(5): | Loop |
| while | while x < 10: | Loop while condition is true |
| break | break | Exit loop |
| continue | continue | Skip current loop |

---

## Data Structures (list / dict)

| Type | Example | Description |
|------|---------|-------------|
| list | a = [1, 2, 3] | Store multiple values |
| dict | d = {"a":1, "b":2} | Key-value data |
| append | a.append(4) | Add item to list |
| pop | a.pop() | Remove item |
| keys | d.keys() | Get keys |
| values | d.values() | Get values |

## File Handling

| Function | Example | Description |
|----------|---------|-------------|
